# AI-website

마우스로 드래그하며 그림을 완성할 수 있는 실험용 드로잉 웹앱입니다.

## 실행 방법

1. 저장소를 클론한 뒤 프로젝트 디렉터리로 이동합니다.
2. 간단하게 정적 서버를 실행하세요.

```bash
python -m http.server 3000
```

3. 브라우저에서 `http://localhost:3000`에 접속하면 캔버스 도구를 이용할 수 있습니다.

## 주요 기능

- 드로잉: 마우스나 터치로 부드러운 선을 그릴 수 있습니다.
- 마지막 선 완성: 시작점과 끝점을 연결해 면을 채울 수 있습니다.
- 캔버스 초기화: 캔버스를 비워 새롭게 그릴 수 있습니다.
- 이미지 저장: 완성한 그림을 PNG 파일로 내려받을 수 있습니다.
